The Basics

Note Up is a dynamic note taking platform that integrates HTML formatting, Latex typesetting, and integrated editors for code and spreadsheets all in one place. The platform also includes various shortcuts and customizable key patterns. The notes can also be printed and saved as a PDF to be used locally.

Key Board Shortcuts

There are various out of box keyboard shortcuts for both HTML and Latex. These can be used by holding Control and Alt while selecting a key. If you are within a Latex frame like \( Latex \) the Latex the set of Latex shortcuts will be used. Otherwise, the HTML shortcuts will be uses. A Latex frame can be placed with Control + Alt + e. For the HTML shortcuts, the tags can be placed around text by highlighting the text then applying the shortcut. Both the Latex and HTML shortcuts can be customized in the settings page.

Other Shortcuts

  • * places a bullet point. Bullet points can be nested with more *'s
  • img:URL places an image from the specified url>
  • cod:LANGUAGE,ID places a code editor of the specified language and tracks the code with the id given
